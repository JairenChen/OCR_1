# AGENTS.md

## 1. 项目定位

本项目是一个“视频字幕与画面文案提取”学习型工程项目。

项目目标不是一开始就追求最复杂模型，而是帮助我从基础逐步掌握视频 OCR / 字幕检测 / 文案提取的完整流程，并最终实现一个可运行、可测试、可扩展的工具。

我的基础情况：

- 会基础 Python
- 会基础 PyTorch 和深度学习
- 对视频处理、OCR、字幕检测、字幕识别、视频文案提取了解较少
- 希望在开发项目的同时学习相关知识

你在本项目中的角色是：

- 编程助手
- 项目架构师
- 学习导师
- 代码审查员
- OCR / 视频理解方向的入门教练

请不要只给代码，也要解释为什么这样设计。

## 2. 总体开发原则

### 2.1 先 MVP，后优化

本项目必须分阶段推进：

1. 先完成最小可运行版本
2. 再逐步优化准确率、速度和鲁棒性
3. 最后再考虑模型训练、YOLO 检测、字幕区域定位、多语言 OCR 等高级功能

不要一开始就引入过多复杂模块。

优先采用如下路线：

```text
视频输入
→ 抽帧
→ 字幕/文字区域检测
→ OCR 识别
→ 文本清洗与去重
→ 时间轴合并
→ 导出 TXT / JSON / SRT
```

## 4. 项目技术路线

### 4.1 第一阶段：规则 + 现成 OCR

第一阶段优先完成可运行 pipeline，不训练模型。

建议实现：

- 使用 OpenCV 或 ffmpeg 读取视频
- 按固定时间间隔抽帧
- 调用现成 OCR 引擎识别画面文字
- 保存每一帧的 OCR 结果
- 对连续重复文本进行合并
- 导出结构化结果

输出格式示例：

```json
[
  {
    "start_time": "00:00:01.200",
    "end_time": "00:00:03.800",
    "text": "example subtitle text",
    "confidence": 0.92
  }
]
```

---

### 4.2 第二阶段：字幕区域过滤

第二阶段重点解决“画面里有很多文字，但我只想提取字幕/文案”的问题。

可以逐步加入：

- 只识别画面下方区域
- 按文字框位置过滤
- 按文字框高度、宽度、面积过滤
- 按置信度过滤
- 合并相邻文本框
- 去除水印、角标、Logo、UI 文案

请在实现时解释每个过滤规则的作用。

---

### 4.3 第三阶段：YOLO 字幕框检测

当基础 pipeline 跑通后，再考虑训练或使用 YOLO 检测字幕区域。

目标：

- 用 YOLO 检测视频画面中的字幕框 / 文案框
- 将检测到的区域裁剪出来
- 再送入 OCR 识别
- 提升复杂场景下的字幕定位能力

请不要默认直接上 YOLO。只有当规则方法不够稳定时，再建议进入该阶段。

---

### 4.4 第四阶段：多语言与复杂场景

后续可支持：

- 中文
- 英文
- 中英混合
- 西班牙语等多语言
- 竖排文字
- 花字字幕
- 带描边字幕
- 动态字幕
- 短视频画面文案
- 电商视频卖点文案提取

请在设计时保留扩展空间。

## 11. 推荐的阶段性任务

请优先按以下顺序推进：

### Milestone 1：视频抽帧

目标：

- 输入视频路径
- 按固定间隔抽帧
- 保存图片帧
- 输出帧时间戳

### Milestone 2：单帧 OCR

目标：

- 对一张图片执行 OCR
- 输出文字、位置框、置信度

### Milestone 3：视频 OCR Pipeline

目标：

- 对视频抽帧
- 对每帧 OCR
- 保存所有识别结果

### Milestone 4：字幕去重与时间轴合并

目标：

- 识别连续帧中重复出现的字幕
- 合并成带 start_time 和 end_time 的文本片段

### Milestone 5：导出 TXT / JSON / SRT

目标：

- 导出纯文本
- 导出结构化 JSON
- 导出字幕 SRT 文件

### Milestone 6：字幕区域过滤

目标：

- 只保留画面下方或指定区域的文字
- 减少水印、Logo、UI 干扰

### Milestone 7：可视化调试

目标：

- 在帧图上画出 OCR 文本框
- 保存 debug 图片
- 方便观察识别错误

### Milestone 8：模型检测增强

目标：

- 评估是否需要 YOLO 检测字幕框
- 如果需要，再设计数据集、标注格式、训练流程和推理流程
